# SerpentLua Plugin index

- The server for SerpentLua's plugin index.

**Full documentation (v1): [Guide](docs/v1.md).**

oh and if you wanna run without reading each and every bit of the source code to find out what the fuck is going on, make sure to add environment variables:

- `DISCORD_PRIV_WEBHOOK` which sends out an embed for every plugin that gets published so index staff can review these plugins
- `DISCORD_PUB_WEBHOOK` which only sends out notifications when a plugin is approved
